Java网络编程

1. 网络编程

  1. 网络编程指的是客户端网络通信（C/S）JavaWeb才是网站开发（B/S）
  2. 网络通信的要素：IP和端口
  3. 网络通信协议：TCP、UDP
  4. Java中的IP地址相关类：InetAddress;IP地址分类：ipv4（32位）和ipv6（128位）（0-9a-f）2001:0bb2:aaaa:0015:0000:0000:1aa1:1312 java中的InetAddress类可以通过域名获取IP，可以通过IP获取主机名称，可以通过IP获取域名
  5. 端口：表示计算机上的一个程序的进程，端口号不可冲突，范围为65536，TCP和UDP都有65536个端口。
    端口分类：
    • 公用端口 0-1023
      • HTTP 80
      • HTTPS 443
      • FTP 21
      • Telnet 23
    • 程序注册端口 1024-49151 （分配用户或程序）
      • Tomcat 8080
      • MySQL 3306
      • Oracle 1521
    • 动态、私有 49152-65535
  6. 通信协议
    网络通信协议：速率、传输码率、代码结构、传输控制
  7. TCP与UDP的比较
    • TCP：类似于打电话
      • 链接，稳定
      • 三次握手、四次挥手
      • 客户端和服务端
      • 传输完成，释放连接，效率低
    • UDP：类似于发短信
      • 不连接，不稳定
      • 客户端和服务端没有明确的界限
      • DDOS 饱和攻击
  8. UDP在客户端新建一个socket，就可以向服务器上的某个端口发送包，而不负责包的接收，就算服务端有问题也不会报错，类似于广播。UDP没有客户端和服务端的概念，都是接收端和发送端。因此UDP可以实现聊天。
  9. URL 统一资源定位符，用于定位互联网上的某个资源
    组成：协议://IP地址:端口号/path_of_file
